# 1. 데이터 처리 부분
## 1.1. 수집된 데이터 로드
* SD카드에서 PC로 가져온 Log파일(.txt)을 한 줄씩 스트링으로 로드
* 각 줄을 기준에 맞추어 parsing하고 배열에 저장
## 1.2. 로드된 데이터 항목별로 나누기
* Parsing 된 배열을 항목별(가속도, 각속도, 지자기 등)로 나누어 저장
* 기왕 나눈김에 txt대신 다른 파일로 저장해도 좋을듯
## 1.3. 나누어진 데이터처리
### 1.3.1. 필터
* MAF
패러미터로 Window 크기를 받음
* LPF
패러미터로 Weight 를 받음
